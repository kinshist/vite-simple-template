# Vite Coding Project Template

通常のWEBサイト制作にViteを導入したテンプレートです。
HTML（ejsライク）・Sass・JSの使用を想定しています。

## Prettier & Lint

VSCode拡張機能で以下を入れておく。
Prettier - Code Formatter
ESLint

## 起動方法

インストール：`npm install`  
開発：`npm run dev`  
ビルド：`npm run build`
